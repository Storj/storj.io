*ВАЖНО* Вы не сможете установить больше одного узла хранения с Вашим одноразовым токеном авторизации. По сравнению с v2 сети больше нет никакого смысла запускать большое количество узлов хранения v3 из одного места. Если Вы имеете 10ТБ места на диске, один узел с 10ТБ предоставленного места будет иметь столько же дохода, как и два узла по 5ТБ. Пожалуйста, прочитайте наш блог,и V3 White Paper для большего понимания.

Если у Вас имеются StorjShare ноды v2, мы просим Вас держать их включенными до V3 Beta.
